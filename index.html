1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS digunakan untuk mengatur tampilan visual dari halaman web. JavaScript digunakan untuk interaktivitas dan membuat halaman menjadi dinamis.

2. Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

CSS:

  • Memisahkan desain dan konten
  • Konsistensi tampilan
  • Responsif dan ringan

JavaScript:

  • Interaktif dan dinamis
  • Manipulasi DOM langsung
  • Validasi form di sisi klien

3. Jelaskan apa saja keuntungan menggunakan Bootstrap?

  • Responsif otomatis
  • Banyak komponen siap pakai
  • Konsistensi tampilan
  • Dokumentasi lengkap

4. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

JavaScript dapat digunakan untuk mengubah gaya CSS secara dinamis.

document.getElementById("myButton").addEventListener("click", function() {
  document.getElementById("myElement").style.color = "red";
  document.getElementById("myElement").className = "highlighted";
});

5. Contoh penggunaan Bootstrap dan fungsinya

Contoh penggunaan Bootstrap:

<h1 class="mb-1">LAPORAN TEORI</h1>
<h2 class="mb-4">Yoga Abdilah | 202332023</h2>

Fungsi: `mb-1` dan `mb-4` adalah kelas Bootstrap untuk mengatur margin bawah (spacing antar elemen).
